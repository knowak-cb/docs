## Method Naming Convention

All methods follow a consistent naming pattern:

```
{visibility}/{action}_{resource}
```

**Examples**:
- `public/get_instruments` - Get list of instruments
- `private/buy` - Place a buy order
- `private/get_account_summary` - Get account information
- `public/subscribe` - Subscribe to data feed

## Request Format

All methods use JSON-RPC 2.0 format:

```json
{
  "jsonrpc": "2.0",
  "method": "public/get_instruments",
  "params": {
    "currency": "BTC",
    "kind": "future"
  },
  "id": 1
}
```

## Response Format

Successful responses include the result:

```json
{
  "jsonrpc": "2.0",
  "result": {
    "data": "..."
  },
  "usIn": 1704153600000000,
  "usOut": 1704153600001234,
  "usDiff": 1234,
  "id": 1
}
```

Error responses include error details:

```json
{
  "jsonrpc": "2.0",
  "error": {
    "code": 10009,
    "message": "invalid_argument",
    "data": {
      "reason": "invalid parameter",
      "param": "currency"
    }
  },
  "id": 1
}
```

## Authentication

### Public Methods

No authentication required:

```bash
curl "https://www.deribit.com/api/v2/public/get_time"
```

### Private Methods

Require access token:

```bash
curl "https://www.deribit.com/api/v2/private/get_account_summary?currency=BTC" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

## Rate Limits

| Method Type | Rate Limit |
|------------|------------|
| Public (non-matching) | 20 requests/second |
| Public (matching engine) | 10 requests/second |
| Private (non-trading) | 20 requests/second |
| Private (trading) | 10 requests/second |

<Warning>
Exceeding rate limits will result in temporary IP bans (1-30 minutes).
</Warning>

## Quick Examples

### Get Server Time

```bash
curl "https://www.deribit.com/api/v2/public/get_time"
```

### Get Instruments

```bash
curl "https://www.deribit.com/api/v2/public/get_instruments?currency=BTC&kind=future"
```

### Get Order Book

```bash
curl "https://www.deribit.com/api/v2/public/get_order_book?instrument_name=BTC-PERPETUAL&depth=10"
```

### Place Order (Authenticated)

```bash
curl "https://www.deribit.com/api/v2/private/buy" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d "instrument_name=BTC-PERPETUAL" \
  -d "amount=10" \
  -d "type=limit" \
  -d "price=50000"
```

### Get Account Summary (Authenticated)

```bash
curl "https://www.deribit.com/api/v2/private/get_account_summary?currency=BTC" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## WebSocket Example

```javascript
const WebSocket = require('ws');
const ws = new WebSocket('wss://www.deribit.com/ws/api/v2');

ws.on('open', function open() {
  // Get server time
  ws.send(JSON.stringify({
    "jsonrpc": "2.0",
    "method": "public/get_time",
    "id": 1
  }));
  
  // Subscribe to order book
  ws.send(JSON.stringify({
    "jsonrpc": "2.0",
    "method": "public/subscribe",
    "params": {
      "channels": ["book.BTC-PERPETUAL.100ms"]
    },
    "id": 2
  }));
});

ws.on('message', function incoming(data) {
  const response = JSON.parse(data);
  console.log(response);
});
```

## Error Handling

Always check for errors in responses:

```javascript
const response = await apiCall();

if (response.error) {
  console.error(`Error ${response.error.code}: ${response.error.message}`);
  // Handle error
} else {
  console.log('Success:', response.result);
}
```
