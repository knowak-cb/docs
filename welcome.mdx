# Welcome to Deribit API Documentation

Deribit is a leading cryptocurrency derivatives exchange offering Bitcoin and Ethereum futures and options trading. This documentation provides comprehensive information about the Deribit API.

## What is Deribit?

Deribit is a cryptocurrency derivatives exchange that specializes in:
- Bitcoin (BTC) futures and options
- Ethereum (ETH) futures and options
- Advanced trading features
- High liquidity and low latency

## API Overview

The Deribit API provides programmatic access to:
- **Market Data**: Real-time and historical market information
- **Trading**: Place, modify, and cancel orders
- **Account Management**: Manage your account settings and preferences
- **Wallet Operations**: Deposits, withdrawals, and transfers
- **Subscriptions**: Real-time data feeds via WebSocket

## API Protocols

Deribit supports two protocols:
- **WebSocket**: For real-time data and low-latency trading
- **HTTP REST**: For standard API requests

Both protocols use JSON-RPC 2.0 format for requests and responses.
